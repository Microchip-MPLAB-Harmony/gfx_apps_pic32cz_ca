# Blank Quickstart

![](./../../docs/html/blank_quickstart.png)

This demonstration provides a starting point to integrate a third-party graphics library with MPLAB Harmony Graphics Suite.

|MPLABX Configuration|Board Configuration|
|:-------------------|:------------------|
